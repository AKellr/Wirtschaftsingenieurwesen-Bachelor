### 🌐 **Key Concepts Covered in Detail**

| **Concept**             | **German Term**          | **Key Information**                                                                                                                                         |
| ----------------------- | ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Technical Drawing**   | **Technische Zeichnung** | Communication and documentation tool; based on **Norms** (DIN, ISO)                                                                                         |
| **Projection Method 1** | **Projektionsmethode 1** | European method; **Top View** [Draufsicht] is placed **below** the **Front View** [Vorderansicht].                                                          |
| **Isometry**            | **Isometrie**            | Axis ratio is **1:1:1**; angles are typically **$30^{\circ}$**.                                                                                             |
| **Cabinet Projection**  | **Kabinett-Projektion**  | A type of Dimetry; axis ratio is **$\frac{1}{2}:1:1$**; angle is typically **$45^{\circ}$**.                                                                |
| **Scale Rule**          | **Maßstab**              | **Dimensions** [Maße] always represent the **true length** [wahre Länge] of the part, regardless of whether the drawing is reduced (1:x) or enlarged (x:1). |
| **Hatching**            | **Schraffur**            | Narrow solid lines, typically at $45^{\circ}$, used to identify cut surfaces in sectional views.                                                            |
