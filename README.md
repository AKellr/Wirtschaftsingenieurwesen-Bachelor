# Wirtschaftsingenieurwesen (B.Sc.) — Modulnotizen

Eine strukturierte Sammlung von persönlichen Lernnotizen, Übungsaufgaben und Zusammenfassungen aus meinem Studium des Wirtschaftsingenieurwesens an der Hochschule Koblenz.

## Zweck des Repositories

Dieses Repository dient als zentrale, digitale Ablage für meine Studienunterlagen. Es dokumentiert meinen Lernfortschritt und hilft mir dabei:
* Vorlesungsinhalte strukturiert aufzubereiten und langfristig zu sichern.
* Übungsaufgaben und anwendungsbezogene Fragestellungen systematisch durchzuarbeiten.
* Eine übersichtliche Wissensdatenbank für die Prüfungsvorbereitung zu pflegen.

---

## Tech Stack & Formatierung

* **Format:** Markdown (`.md`) / Obsidian-kompatibel (für einfache Navigation und Verknüpfung der Notizen).
* **Struktur:** Modular aufgebaut nach Vorlesungswochen und Themenschwerpunkten.
