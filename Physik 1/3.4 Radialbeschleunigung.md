# 🚀 3.4. Radialbeschleunigung ($a_r$)

## 1. Definition und Funktion

- Die **Radialbeschleunigung** ($a_r$) tritt bei der Bewegung eines Körpers auf einer **Kreisbahn** auf.
    
- Ihre Hauptfunktion ist die Erzeugung einer **seitlichen Beschleunigung**, die die Ablenkung von der geradlinigen Bewegung bewirkt.
    
- **Wichtig:** Die Radialbeschleunigung hat **keinen Einfluss auf den Betrag der Geschwindigkeit** ($v$). Sie ändert lediglich die Richtung des Geschwindigkeitsvektors ($\vec{v}$).
    

---

## 2. Richtung und Wirkung

- **Richtung:** Die Radialbeschleunigung ($\vec{a}_r$) ist **auf den Mittelpunkt** des Kreises gerichtet (daher auch **Zentripetalbeschleunigung** genannt).
    
- Sie ist **antiparallel** zum Radiusvektor ($\vec{r}$), da der Radiusvektor vom Mittelpunkt weg zeigt.
    
- **Wirkung:** Die Radialbeschleunigung ($\vec{a}_r$) wirkt **senkrecht zur Bahngeschwindigkeit** ($\vec{v}$) und damit senkrecht zur Bahnkurve.
    

---

## 3. Mathematischer Betrag

Der Betrag der Radialbeschleunigung ($a_r$) wird durch folgende Formeln ausgedrückt:

### 3.1. Bezogen auf die Bahngeschwindigkeit ($v$)

$$\vec{a}_r = \frac{v^2}{r}.$$

### 3.2. Bezogen auf die Winkelgeschwindigkeit ($\omega$)

$$a_r = \omega^2 r.$$

| **Variable** | **Bedeutung**         | **Einheit** |     |
| ------------ | --------------------- | ----------- | --- |
| $$r$$        | Radius der Kreisbahn  | $$[m]$$     |     |
| $$v$$        | Bahngeschwindigkeit   | $$[m/s]$$   |     |
| $$\omega$$   | Winkelgeschwindigkeit | $$[rad/s]$$ |     |
|              |                       |             |     |
|              |                       |             |     |

> **💡 Herleitungshinweis:** Die Formel wird hergeleitet durch die Division der Geschwindigkeitsänderung ($\Delta v$) durch die Zeit ($\Delta t$). Im Grenzfall kleiner Intervalle ($\Delta t \to 0$) geht $\frac{\Delta v}{\Delta t}$ in die Beschleunigung ($a$) über und $\frac{\Delta s}{\Delta t}$ in die Geschwindigkeit ($v$). Die Beziehung $\Delta v = v \cdot \frac{\Delta s}{r}$ wird dabei genutzt.

---

## 4. Physikalische Anwendung (Beispiel: Erdsatellit)

- Bei **Erdsatelliten** wird die benötigte **Radialbeschleunigung** ($a_r$) durch die **Gravitationskraft** (Anziehkraft der Erde) bereitgestellt.
    
- Der Satellit wird ständig **zur Erde hin beschleunigt**, wodurch seine Flugbahn von einer Geraden abgelenkt wird und die **Kreis- oder Ellipsenbahn** entsteht.
